Notes on automatic differentiation¶

Preliminaries¶

Definition of normalised derivative:

(1)¶\[x^{\left[ n\right]}\left( t \right) = \frac{1}{n!} x^{\left( n\right)}\left( t \right).\]

General Leibniz rule: given \(a\left( t \right) = b\left( t \right) c\left( t \right)\), then

(2)¶\[a^{\left[ n\right]}\left( t \right) = \sum_{j=0}^n b^{\left[ n - j\right]}\left( t \right) c^{\left[ j\right]}\left( t \right).\]

Basic arithmetic¶

Addition and subtraction¶

Given \(a\left( t \right) = b\left( t \right) \pm c\left( t \right)\), trivially

(3)¶\[a^{\left[ n \right]}\left( t \right) = b^{\left[ n \right]}\left( t \right) \pm c^{\left[ n \right]}\left( t \right).\]

Inverse trigonometric functions¶

Inverse sine¶

Given \(a\left( t \right) = \arcsin b\left( t \right)\), we have

(4)¶\[a^\prime\left( t \right) = \frac{b^\prime\left( t \right)}{\sqrt{1 - b^2\left( t \right) }},\]

or, equivalently,

(5)¶\[a^\prime\left( t \right) \sqrt{1 - b^2\left( t \right) } = b^\prime\left( t \right).\]

We introduce the auxiliary function

(6)¶\[c\left( t \right) = \sqrt{1 - b^2\left( t \right) },\]

so that (5) can be rewritten as

(7)¶\[a^\prime\left( t \right) c\left( t \right) = b^\prime\left( t \right).\]

Applying the normalised derivative of order \(n-1\) to both sides yields, via (1):

(8)¶\[\left[a^\prime\left( t \right) c\left( t \right)\right]^{\left[ n - 1 \right]} = n b^{\left[ n \right]} \left( t \right).\]

We can now apply the general Leibniz rule (2) to the left-hand side and re-arrange the terms to obtain, for \(n > 0\):

(9)¶\[a^{\left[ n \right]}\left( t \right) = \frac{1}{n c^{\left[ 0 \right]}\left( t \right)}\left[ n b^{\left[ n \right]}\left( t \right) - \sum_{j=1}^{n-1} j c^{\left[ n - j \right]}\left( t \right) a^{\left[ j \right]}\left( t \right) \right].\]

Inverse cosine¶

The derivation is identical to the inverse sine, apart from a sign change. Given \(a\left( t \right) = \arccos b\left( t \right)\), the final result is, for \(n > 0\):

(10)¶\[a^{\left[ n \right]}\left( t \right) = -\frac{1}{n c^{\left[ 0 \right]}\left( t \right)}\left[ n b^{\left[ n \right]}\left( t \right) + \sum_{j=1}^{n-1} j c^{\left[ n - j \right]}\left( t \right) a^{\left[ j \right]}\left( t \right) \right],\]

with \(c\left( t \right)\) defined as:

(11)¶\[c\left( t \right) = \sqrt{1 - b^2\left( t \right) }.\]

Inverse tangent¶

Given \(a\left( t \right) = \arctan b\left( t \right)\), we have

(12)¶\[a^\prime\left( t \right) = \frac{b^\prime\left( t \right)}{1 + b^2\left( t \right) },\]

or, equivalently,

(13)¶\[a^\prime\left( t \right) \left[1 + b^2\left( t \right) \right] = b^\prime\left( t \right).\]

We introduce the auxiliary function

(14)¶\[c\left( t \right) = b^2\left( t \right),\]

so that (13) can be rewritten as

(15)¶\[a^\prime\left( t \right) + a^\prime\left( t \right) c\left( t \right) = b^\prime\left( t \right).\]

Applying the normalised derivative of order \(n-1\) to both sides yields, via (1) and (3):

(16)¶\[n a^{\left[ n \right]} \left( t \right) + \left[a^\prime\left( t \right) c\left( t \right)\right]^{\left[ n - 1 \right]} = n b^{\left[ n \right]} \left( t \right).\]

With the help of the general Leibniz rule (2), after re-arranging we obtain, for \(n > 0\):

(17)¶\[a^{\left[ n \right]}\left( t \right) = \frac{1}{n \left[ c^{\left[ 0 \right]}\left( t \right) + 1 \right]}\left[ n b^{\left[ n \right]}\left( t \right) - \sum_{j=1}^{n-1} j c^{\left[ n - j \right]}\left( t \right) a^{\left[ j \right]}\left( t \right) \right].\]
